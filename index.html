Summary: Latitude vs X

This website is the findings from a previous assignment, in which we were meant to create a Python script to visualise the weather of over 500 cities of varying distances from the equator. This was done by using a Python library and the OpenWeatherMap API. Once the data was retrieved, four scatter graphs were created, in which we compared the maximum temperature, humidity, cloudiness and wind speed to the latitude. These visualisations then help us find if there are any obvious patterns.
